# Cybersecurity Awareness Game

## Overview

The Cybersecurity Awareness Game is an educational web-based application designed to improve cybersecurity knowledge through an interactive quiz experience.

The game provides age-appropriate cybersecurity questions, real-time feedback, score tracking, leaderboards, and achievement certificates to help users learn safe online practices in an engaging way.

---

## Features

### Interactive Quiz
- Multiple cybersecurity questions.
- Age-based question categories.
- Multiple-choice answers.

### Real-Time Feedback
- Instant answer validation.
- Educational cybersecurity tips.

### Progress Tracking
- Progress bar.
- Score calculation.
- Performance monitoring.

### Leaderboard
- Top player rankings.
- Score comparison.

### Certificate Generation
- Achievement certificate for successful completion.

### Cybersecurity Education
Topics include:
- Password Security
- Phishing Attacks
- Malware
- Ransomware
- VPN Usage
- Two-Factor Authentication (2FA)
- Safe Browsing Practices
- Public Wi-Fi Risks
